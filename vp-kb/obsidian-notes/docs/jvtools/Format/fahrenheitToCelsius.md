<h3>
  <code>Format.fahrenheitToCelsius(fahrenheit)</code>
</h3>
华氏度转摄氏度


#### 添加版本

1.0.7

#### 参数

1. `fahrenheit` *(Number)*: 华氏度。

#### 返回

*(String)*: 转化后的摄氏度。

#### 例子

```javascript
Format.Format.celsiusToFahrenheit(104)
// => 40
```
